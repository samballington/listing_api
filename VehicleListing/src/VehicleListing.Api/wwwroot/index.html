Endpoints

Method Route Description
GET /api/vehicles List vehicles — supports filtering by make, model, year, price, status
GET /api/vehicles/{id} Get vehicle by ID
POST /api/vehicles Create a vehicle
PUT /api/vehicles/{id} Update a vehicle
DELETE /api/vehicles/{id} Delete a vehicle — returns 204
GET /api/vehicles/search?query= Full-text search across make, model, description
GET /api/dealers List all dealers
GET /api/dealers/{id} Get dealer by ID — includes their vehicles
POST /api/dealers Create a dealer
